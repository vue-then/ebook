# ebook
## https://github.com/futurepress/epub.js

### epubjs核心类介绍
- Book 阅读器的解析
- Rendition 阅读器的渲染
- Locations 阅读器的定位
- Navigation 阅读器的目录
- View Manageer 阅读器视图管理
- EpubCFI 文字级别的定位 定位电子书中的任意一个字符
- Theme 管理场景切换
- Spine 负责指定阅读顺序和Section
- Section 指向一个具体的章节 全文检索 和章节切换
- Contents 管理一个章节中的全部资源内容 
- Hook 钩子函数 生命周期
- Annotations 管理标签 文字的高亮显示
